# 🧩 스도쿠 게임 프로젝트

스도쿠는 가로, 세로, 그리고 3×3의 작은 정사각형 안에서 1부터 9까지의 숫자를 중복 없이 채워 넣는 논리 퍼즐입니다.  
주어진 힌트와 숫자의 규칙을 기반으로 퍼즐을 해결해야 합니다.

---

## 📜 게임 규칙

- 각 **행(row)**에는 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.  
- 각 **열(column)**에도 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.  
- 각 **3×3 박스**에도 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.

---

## 🛠️ 주요 함수 설명

| 함수 이름           | 기능 설명 |
|--------------------|-----------|
| `initialize_board` | 빈 스도쿠 보드를 생성합니다 |
| `print_board`      | 현재 스도쿠 보드를 출력합니다 |
| `solve_sudoku`     | 주어진 스도쿠 퍼즐을 해결합니다 |
| `generate_sudoku`  | 빈 칸이 포함된 완성된 스도쿠 퍼즐을 생성합니다 |
| `play_sudoku`      | 사용자가 스도쿠 게임을 플레이할 수 있도록 합니다. 잘못된 입력 또는 퍼즐 완성 시 게임이 종료됩니다 |
